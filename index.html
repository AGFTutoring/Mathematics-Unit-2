Proof

Mathematical proof establishes that a statement is true with absolute certainty. At P2 level you must be able to use proof by deduction and proof by exhaustion, and understand disproof by counter-example.

Proof by Deduction
Start from known facts, definitions, or previously proven results and use logical steps to reach the conclusion. This is the most common form of proof at this level.
Proof by Exhaustion
Check every possible case. Only practical when there is a finite and small number of cases to verify.
Disproof by Counter-example
To disprove a statement, find a single example where it fails. One counter-example is sufficient to disprove a universal claim.
Proof by Contradiction
Assume the statement is false, then show this leads to a logical contradiction. Therefore the original statement must be true (e.g. proof that √2 is irrational).
Common exam question: "Prove that the sum of two consecutive odd numbers is divisible by 4." Let the numbers be (2n − 1) and (2n + 1). Sum = 4n. Since 4n = 4 × n, the sum is divisible by 4. ∎
Exam tip: When proving algebraic identities, always work from one side only and show it equals the other. Never work on both sides simultaneously — examiners want to see a clear chain of reasoning.
02

Algebra & Functions

P2 extends algebraic techniques to include the factor theorem, algebraic division, and partial fractions.

Factor Theorem & Remainder Theorem

Factor Theorem
If f(a) = 0 then (x − a) is a factor of f(x). Conversely, if (x − a) is a factor then f(a) = 0.
Remainder Theorem
When f(x) is divided by (x − a), the remainder is f(a).

Algebraic Division

To divide a polynomial f(x) by a linear or quadratic divisor, use either long division or compare coefficients. The result is expressed as:

Division identity
f(x) = q(x) × divisor + remainder

Partial Fractions

Express a proper algebraic fraction as a sum of simpler fractions:

Denominator typePartial fraction form
Linear factors: (ax+b)(cx+d)A/(ax+b) + B/(cx+d)
Repeated factor: (ax+b)²A/(ax+b) + B/(ax+b)²
Three linear: (ax+b)(cx+d)(ex+f)A/(ax+b) + B/(cx+d) + C/(ex+f)
Exam tip: To find constants quickly, use the "cover-up" method — substitute the value of x that makes each factor zero. For repeated factors, you'll still need to compare coefficients for the extra constant.
03

Coordinate Geometry

P2 extends coordinate geometry to parametric equations and more advanced curve work.

Parametric Equations

A curve can be described by expressing x and y separately in terms of a parameter t (or θ). For example, x = t², y = 2t describes a parabola.

Converting to Cartesian
Eliminate the parameter by expressing t in terms of x or y from one equation, then substituting into the other.
Finding intersections
Substitute the parametric equations into the equation of the line or curve and solve for t.
Common exam question: A curve is defined by x = 3t, y = 6/t. Find the Cartesian equation. From x = 3t → t = x/3. Substitute: y = 6/(x/3) = 18/x. So xy = 18.

Sequences & Series

A sequence is an ordered list of terms. A series is the sum of the terms of a sequence. P2 covers arithmetic and geometric sequences and their sums, including convergence of infinite geometric series and the binomial expansion.

Arithmetic Sequences & Series

nth term
uₙ = a + (n − 1)d
Sum of n terms
Sₙ = n/2 [2a + (n − 1)d]
Alternative sum
Sₙ = n/2 (a + l)   where l = last term

Geometric Sequences & Series

nth term
uₙ = arⁿ⁻¹
Sum of n terms
Sₙ = a(1 − rⁿ) / (1 − r)  (r ≠ 1)
Sum to infinity (|r| < 1)
S∞ = a / (1 − r)
Watch out: The sum to infinity only exists when |r| < 1 (the series converges). If |r| ≥ 1, the series diverges and has no finite sum. Always state the condition for convergence.

Binomial Expansion

Binomial theorem (positive integer n)
(a + b)ⁿ = Σ ⁿCᵣ aⁿ⁻ʳ bʳ   for r = 0 to n
Binomial coefficient
ⁿCᵣ = n! / [r!(n − r)!]
Exam tip: Use Pascal's triangle for small values of n. For larger n or when only specific terms are needed, use the formula ⁿCᵣ directly. Remember to include powers of both a and b in each term.

Sigma Notation & Recurrence Relations

  • Sigma notation: Σ provides a compact way to write a sum. E.g. Σ(r=1 to n) r² means 1² + 2² + ... + n².
  • Recurrence relation: Defines each term as a function of previous terms. E.g. uₙ₊₁ = 3uₙ − 2 with u₁ = 4.
Common exam question: The 3rd term of a GP is 24 and the 6th term is 192. Find a and r. ar² = 24 and ar⁵ = 192. Dividing: r³ = 8, so r = 2. Then a(4) = 24, so a = 6.

Exponentials & Logarithms

Logarithms are the inverse of exponentials. If aˣ = b then log_a(b) = x. This section covers the laws of logarithms, solving exponential equations, the natural logarithm, and modelling with exponentials.

Laws of Logarithms

Product rule
log_a(xy) = log_a(x) + log_a(y)
Quotient rule
log_a(x/y) = log_a(x) − log_a(y)
Power rule
log_a(xⁿ) = n log_a(x)
Change of base
log_a(x) = log_b(x) / log_b(a)
Special values
log_a(a) = 1  |  log_a(1) = 0
Inverse relationship
a^(log_a(x)) = x  |  log_a(aˣ) = x

The Natural Logarithm & Exponential

The natural exponential function y = eˣ has base e ≈ 2.718. Its inverse is y = ln x. The key property: ln(eˣ) = x and e^(ln x) = x.

Differentiation
d/dx [eˣ] = eˣ
Integration
∫ eˣ dx = eˣ + c

Solving Exponential Equations

  • To solve aˣ = b: take logarithms of both sides → x = log(b) / log(a).
  • For equations like 2²ˣ − 5(2ˣ) + 6 = 0: substitute y = 2ˣ to get a quadratic in y.

Modelling with Exponentials

Many real-world processes follow exponential models of the form y = Aekt (growth if k > 0, decay if k < 0) or y = Abt. To find constants from data, take logarithms to reduce to a linear relationship:

Reducing y = axⁿ to linear form
log y = n log x + log a  →  plot log y against log x (gradient n, intercept log a)
Reducing y = abˣ to linear form
log y = x log b + log a  →  plot log y against x (gradient log b, intercept log a)
Common exam question: The population of bacteria is modelled by P = 200e0.03t. Find when P = 1000. 1000 = 200e0.03t → e0.03t = 5 → 0.03t = ln 5 → t = ln 5 / 0.03 ≈ 53.6.

Trigonometry

P2 trigonometry builds on P1 with compound angle formulae, double angle formulae, and the R-formula for combining sine and cosine functions.

Compound Angle Formulae

sin(A ± B)
sin A cos B ± cos A sin B
cos(A ± B)
cos A cos B ∓ sin A sin B
tan(A ± B)
(tan A ± tan B) / (1 ∓ tan A tan B)

Double Angle Formulae

sin 2A
2 sin A cos A
cos 2A (three forms)
cos²A − sin²A = 2cos²A − 1 = 1 − 2sin²A
tan 2A
2 tan A / (1 − tan²A)

The R-formula

Expressions of the form a sin θ ± b cos θ can be written as R sin(θ ± α) or R cos(θ ∓ α), where:

R value
R = √(a² + b²)
α value
tan α = b/a  (0 < α < π/2)

The R-formula is used to find maximum/minimum values of the expression and to solve equations of the form a sin θ + b cos θ = c.

Exam tip: When using the R-formula, expand R sin(θ + α) using the compound angle formula and compare coefficients of sin θ and cos θ. The maximum value of R sin(θ + α) is R (when sin(θ + α) = 1).

Differentiation

P2 extends differentiation to include the chain rule, product rule, quotient rule, and implicit and parametric differentiation.

Key Rules

Chain rule
dy/dx = dy/du × du/dx
Product rule
d/dx[uv] = u(dv/dx) + v(du/dx)
Quotient rule
d/dx[u/v] = [v(du/dx) − u(dv/dx)] / v²

Implicit Differentiation

When y is defined implicitly (e.g. x² + y² = 25), differentiate each term with respect to x, applying the chain rule to terms containing y (multiply by dy/dx), then rearrange for dy/dx.

Common exam question: Find dy/dx for x² + y² = 25. Differentiate: 2x + 2y(dy/dx) = 0 → dy/dx = −x/y.

Integration by Substitution

Replace a complicated expression with a single variable u. Steps: choose u, find du/dx, replace all x terms and dx, integrate in u, then substitute back.

Integration Using Partial Fractions

Split rational functions into partial fractions (Section 02), then integrate each term separately. This often produces logarithmic terms.

Integration of Trigonometric Expressions

Use identities to simplify before integrating:

  • ∫ sin²x dx: use cos 2x = 1 − 2sin²x → sin²x = ½(1 − cos 2x)
  • ∫ cos²x dx: use cos 2x = 2cos²x − 1 → cos²x = ½(1 + cos 2x)
  • ∫ tan²x dx: use tan²x = sec²x − 1
Exam tip: For substitution questions, you are usually told which substitution to use. Write du/dx clearly, change the limits if the integral is definite, and don't forget to convert dx into du.
